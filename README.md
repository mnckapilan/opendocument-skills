# odf-skills

Claude Code skills for working with OpenDocument Format (ODF) files.

## Skills

### `ods` — OpenDocument Spreadsheet

Read and write `.ods` spreadsheet files. Supports:

- Exploring files (sheet names, dimensions)
- Reading rows with pagination
- Getting and setting individual cells
- Appending rows
- Creating files, adding, renaming, and deleting sheets

## Installation

Requires [uv](https://docs.astral.sh/uv/) and [Claude Code](https://claude.ai/code).

Copy the skill directory into your Claude skills folder:

```bash
cp -r ods ~/.claude/skills/ods
```

Then use it in Claude Code:

```
/ods what's in my budget.ods?
```

## Requirements

- [uv](https://docs.astral.sh/uv/) — install with `curl -LsSf https://astral.sh/uv/install.sh | sh`
- Python 3.11+
- [odfpy](https://github.com/eea/odfpy) — installed automatically by `uv` on first use

## Caveats

- `set-cell` rebuilds the sheet element, so cell-level formatting (fonts, colours, borders) is not preserved
- Formulas are replaced by their last computed values when a sheet is rebuilt
- Fixture files in `evals/files/` were generated by the script itself — for more thorough eval coverage, supplement with ODS files exported from LibreOffice or Excel
